## Descrição do Projeto
A **Pokédex Aleatória** é um aplicativo desenvolvido em Flutter que consome a PokéAPI para exibir informações completas sobre Pokémon.  
O usuário pode pesquisar por nome ou número, gerar um Pokémon aleatório, visualizar seus tipos, habilidades, status, descrição oficial e linha de evolução.

O projeto foi estruturado com foco em:
- Integração com API  
- Organização modular  
- Interface intuitiva  
- Estrutura escalável com divisão clara entre páginas, serviços, widgets e modelos
